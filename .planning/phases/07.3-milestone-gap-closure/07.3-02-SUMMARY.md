## Decisions Made

- nil prevRow is mandatory when decoding OldTuple — OldTuple is the complete prior-state row sent by Postgres; passing a TOAST-cached prevRow would overwrite correct prior-column values with post-change cached values, corrupting the before-image
- OldTuple nil guard is required — REPLICA IDENTITY DEFAULT UPDATE messages do not include OldTuple; without the guard, handleUpdate would panic on nil dereference for default-identity tables
