<objective>
Wire EventsDelivered Prometheus metric into StdoutWriter so the default output mode reports delivery metrics.

Purpose: OBS-01 requires kaptanto_events_delivered_total to be non-zero after events flow through any output mode. SSE and gRPC consumers already increment this counter; StdoutWriter (the default mode) never does — leaving the metric permanently zero for most deployments.
Output: StdoutWriter gains a SetMetrics setter (matching the post-construction injection pattern) and increments EventsDelivered after each successful encode. runPipeline passes the shared metrics instance to the writer.
</objective>
