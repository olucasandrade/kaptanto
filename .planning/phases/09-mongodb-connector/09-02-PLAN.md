MongoDB Change Stream document structure (bson.Raw fields):
```
operationType  string  — "insert", "update", "delete", "replace"
ns.db          string  — database name
ns.coll        string  — collection name
documentKey    bson.Raw — {"_id": <value>} — always present
fullDocument   bson.Raw — present for insert, update (with updateLookup), replace; nil for delete
fullDocumentBeforeChange bson.Raw — present when changeStreamPreAndPostImages enabled
clusterTime    bson.Timestamp — event wall-clock time
_id            bson.Raw — resume token document
```

IdempotencyKey format (matching Postgres connector pattern):
```
"<sourceID>:<db>.<collection>:<_id_hex>:<operation>:<clusterTime>"
```

    **Step 2: Implement normalizer.go**

    Package: `internal/parser/mongodb`

    ```go
    // NormalizeChangeEvent converts a raw MongoDB Change Stream event (bson.Raw)
    // into a unified ChangeEvent. The sourceID is embedded in the idempotency key.
    // idGen produces the ULID event ID.
    func NormalizeChangeEvent(raw bson.Raw, sourceID string, idGen *event.IDGenerator) (*event.ChangeEvent, error)
    ```

    Implementation approach:
    1. Decode operationType, ns.db, ns.coll, documentKey, fullDocument, fullDocumentBeforeChange, clusterTime, _id (resume token) from raw using bson.Raw.Lookup() or unmarshaling into a typed struct.
    2. Map operationType to event.Operation: "insert"→OpInsert, "update"→OpUpdate, "delete"→OpDelete, "replace"→OpUpdate. Return error for all others.
    3. Validate: insert with no fullDocument → return error.
    4. Serialize Key: bson.MarshalExtJSON(documentKey, true, false) → json.RawMessage
    5. Serialize After: fullDocument != nil → bson.MarshalExtJSON(fullDocument, true, false); else nil (JSON null via json.RawMessage(nil))
    6. Serialize Before: fullDocumentBeforeChange != nil → bson.MarshalExtJSON; else nil
    7. Timestamp: clusterTime is bson.Timestamp type — use primitive.Timestamp{T: seconds, I: ordinal}. seconds → time.Unix(int64(t.T), 0).UTC()
    8. IdempotencyKey: fmt.Sprintf("%s:%s.%s:%s:%s:%08X%08X", sourceID, db, coll, string(keyJSON), op, clusterTime.T, clusterTime.I)
    9. ResumeToken: raw.Lookup("_id").Value is the resume token bson.Raw; convert to string via hex.EncodeToString or bson.Raw.String().
    10. Metadata: map[string]any{"resume_token": tokenStr, "snapshot": false, "db": db, "collection": coll}

    Use `go.mongodb.org/mongo-driver/v2/bson` and `go.mongodb.org/mongo-driver/v2/bson/primitive`.
