    - Each document is converted to OpRead ChangeEvent via NormalizeChangeEvent (with operation overridden to OpRead)
    - Before appending each row, WatermarkChecker.ShouldEmit(ctx, coll, key, snapshotLSN) is checked; rows that return false are skipped

    - MongoSnapshot accepts: Config (collections, database, sourceID), eventLog eventlog.EventLog, wc *backfill.WatermarkChecker, idGen *event.IDGenerator, appendFn func(ctx, *event.ChangeEvent) error
    - On context cancellation, Run returns context.Canceled
  </behavior>

    ```go
        SourceID    string

    func NewMongoSnapshot(cfg SnapshotConfig, client *mongo.Client, wc *backfill.WatermarkChecker, idGen *event.IDGenerator, appendFn func(context.Context, *event.ChangeEvent) error) *MongoSnapshot

    func (s *MongoSnapshot) Run(ctx context.Context) error
    ```

    Implementation:
    - captureSnapshotLSN: run `db.runCommand({hello:1})` to get clusterTime; encode as uint64(clusterTime.T)<<32 | uint64(clusterTime.I)
    - For each collection: call s.findFn with sort {_id: 1} (no limit — keyset cursor is implicit since _id is always unique and we iterate all results)

    - Return context.Canceled on ctx.Done()

    ```go
    // SourceType returns the detected source database type based on the DSN prefix.

    func (c *Config) SourceType() string {
        if strings.HasPrefix(c.Source, "mongodb://") || strings.HasPrefix(c.Source, "mongodb+srv://") {
    ```
