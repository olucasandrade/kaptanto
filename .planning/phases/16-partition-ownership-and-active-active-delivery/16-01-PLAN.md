**Schema SQL constant** (embed in file, executed by OpenPartitionStore):
```sql
CREATE TABLE IF NOT EXISTS kaptanto_partitions (
    partition_id  INTEGER     PRIMARY KEY,
    owner_node_id TEXT        NULL,
    epoch         BIGINT      NOT NULL DEFAULT 0,
    claimed_at    TIMESTAMPTZ NULL
);
INSERT INTO kaptanto_partitions (partition_id)
SELECT generate_series(0, 63)
ON CONFLICT DO NOTHING;
```

**PartitionClaim** struct:
```go
type PartitionClaim struct {
    PartitionID uint32
    Epoch       int64
}
```

**PartitionStore** struct:
```go
type PartitionStore struct {
    conn   *pgx.Conn
    nodeID string
    mu     sync.RWMutex
    epochs map[uint32]int64 // partitionID → epoch held by this node
}
```

**OpenPartitionStore(ctx, dsn, nodeID string) (*PartitionStore, error)**:
- Connect via pgx.Connect
- Execute createPartitionsTableSQL (CREATE TABLE + INSERT seed)
- Return initialized PartitionStore

**ClaimUnclaimed(ctx context.Context) ([]PartitionClaim, error)**:
- SELECT partition_id FROM kaptanto_partitions WHERE owner_node_id IS NULL ORDER BY partition_id
- For each unclaimed pid, run:
  ```sql
  UPDATE kaptanto_partitions
  SET owner_node_id = $1, epoch = epoch + 1, claimed_at = NOW()
  WHERE partition_id = $2 AND owner_node_id IS NULL
  RETURNING partition_id, epoch
  ```
  Use `conn.QueryRow(...).Scan(...)` — treat `pgx.ErrNoRows` as race loss (skip, no error)
- Update `ps.epochs[claimedID] = epoch` under mu.Lock
- Return non-nil slice of successfully claimed PartitionClaim values

**StealStalePartitions(ctx context.Context, staleNodeID string) ([]PartitionClaim, error)**:
- Run single query:
  ```sql
  UPDATE kaptanto_partitions
  SET owner_node_id = $1, epoch = epoch + 1, claimed_at = NOW()
  WHERE owner_node_id = $2
  RETURNING partition_id, epoch
  ```
- Collect all returned rows into []PartitionClaim
- Update ps.epochs for each stolen partition under mu.Lock
- Return non-nil slice

**ReleaseAll(ctx context.Context) error**:
- Run:
  ```sql
  UPDATE kaptanto_partitions
  SET owner_node_id = NULL, claimed_at = NULL
  WHERE owner_node_id = $1
  ```
- Clear ps.epochs under mu.Lock

**ListOwned(ctx context.Context) ([]PartitionClaim, error)**:
- SELECT partition_id, epoch FROM kaptanto_partitions WHERE owner_node_id = $1 ORDER BY partition_id
- Return non-nil slice

**EpochFor(partitionID uint32) (int64, bool)**:
- Read from ps.epochs under RLock; return (epoch, true) if present, (0, false) if not

**Close(ctx context.Context) error**: close conn

All SQL constants defined as package-level `const` strings at top of file, same style as `membership.go`.
