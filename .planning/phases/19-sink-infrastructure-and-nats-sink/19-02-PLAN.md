import (
    "bytes"

var _ router.Consumer = (*NATSSinkConsumer)(nil)

type NATSSinkConsumer struct {
    id       string
    nc       *natsgo.Conn
    js       jetstream.JetStream
    subjectT *template.Template
    m        *observability.KaptantoMetrics

**NewNATSSinkConsumer:**
1. Parse SubjectTemplate via `template.New("subject").Parse(cfg.SubjectTemplate)` — return error if parse fails.
2. Build nats.Options slice: always include `natsgo.Name("kaptanto-sink")`, `natsgo.MaxReconnects(-1)`, `natsgo.ReconnectWait(2*time.Second)`, `natsgo.ReconnectJitter(500*time.Millisecond, 2*time.Second)`.
3. If cfg.TLS.CAFile != "": append `natsgo.RootCAs(cfg.TLS.CAFile)`.
4. If cfg.TLS.CertFile != "" AND cfg.TLS.KeyFile != "": append `natsgo.ClientCert(cfg.TLS.CertFile, cfg.TLS.KeyFile)`.
5. Connect: `nc, err := natsgo.Connect(cfg.URL, opts...)` — return wrapped error on failure.
6. `js, err := jetstream.New(nc)` — close nc and return error on failure.
7. If cfg.StreamName != "": call `js.Stream(ctx, cfg.StreamName)` with a 5-second timeout context. If error: close nc, return `fmt.Errorf("nats sink: stream %q does not exist — create it before starting kaptanto: %w", cfg.StreamName, err)`.
8. Return `&NATSSinkConsumer{id: id, nc: nc, js: js, subjectT: tmpl}`.

**SetMetrics:** `func (c *NATSSinkConsumer) SetMetrics(m *observability.KaptantoMetrics) { c.m = m }`

**Deliver:**
1. Execute template into buffer: `if err := c.subjectT.Execute(&buf, entry.Event); err != nil { return fmt.Errorf("nats sink: subject template: %w", err) }`.
2. Validate subject: `if !natsgo.IsValidSubject(subject) { return fmt.Errorf("nats sink: invalid subject %q", subject) }`.
3. Marshal event to JSON: `data, err := json.Marshal(entry.Event)`.
4. Build `msg := &natsgo.Msg{Subject: subject, Data: data, Header: natsgo.Header{"Kaptanto-Idempotency-Key": []string{entry.Event.IdempotencyKey}}}`.
5. Record `start := time.Now()`.
6. `_, err = c.js.PublishMsg(ctx, msg)`.
7. If metrics set: `c.m.QueuePublishLatency.WithLabelValues("nats").Observe(time.Since(start).Seconds())`.
8. On error: increment `c.m.QueuePublishErrors` (if set), return wrapped error.
9. On success: increment `c.m.QueuePublishTotal` (if set), return nil.

**Ping:** `if !c.nc.IsConnected() { return fmt.Errorf("nats: not connected (status: %v)", c.nc.Status()) }; return nil`

**Close:** `func (c *NATSSinkConsumer) Close() { c.nc.Close() }`
