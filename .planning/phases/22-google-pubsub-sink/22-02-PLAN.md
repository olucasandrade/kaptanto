<!-- pubsub/v2 API — v2 only, NOT v1 patterns -->
```go
// Import: cloud.google.com/go/pubsub/v2
client, err := pubsub.NewClient(ctx, projectID, clientOpts...)
publisher := client.Publisher(topicID)       // v2 API — NOT client.Topic()
publisher.EnableMessageOrdering = true        // MUST set before first Publish
result := publisher.Publish(ctx, &pubsub.Message{
    Data:        data,
    OrderingKey: orderingKey,
    Attributes:  map[string]string{"Kaptanto-Idempotency-Key": key},
})
serverID, err := result.Get(ctx)             // blocks until broker ack (CHK-01)

// Ordering-key recovery — detect with errors.As, NOT string matching
var paused pubsub.ErrPublishingPaused
if errors.As(err, &paused) {
    publisher.ResumePublish(paused.OrderingKey) // call BEFORE returning error
}

// Ping — use TopicAdminClient.GetTopic
_, err = client.TopicAdminClient.GetTopic(ctx, &pubsubpb.GetTopicRequest{
    Topic: fmt.Sprintf("projects/%s/topics/%s", projectID, topicID),
})

// Close — publisher.Stop() FIRST, then client.Close()
publisher.Stop()
client.Close()
```

    Package name: pubsubsink (matches natssink/sqssink/kafkasink convention)

    File: internal/output/pubsub/consumer.go

    Package doc comment covers CHK-01, DLV-02, DLV-04, DLV-03 design decisions (copy structure from kafkasink package doc).

    Struct:
    ```go
    type PubSubSinkConsumer struct {
        id        string
        client    *pubsub.Client
        publisher *pubsub.Publisher
        projectID string
        topicID   string
        topicT    *template.Template // nil when TopicTemplate is empty
        m         *observability.KaptantoMetrics
    }
    ```

    Constructor (NewPubSubSinkConsumer):
    1. Parse TopicTemplate if non-empty; return error for invalid template
    2. Build clientOpts: if CredentialsFile != "", append option.WithCredentialsFile(path)
    3. Call pubsub.NewClient(context.Background(), cfg.ProjectID, clientOpts...)
    4. Call client.Publisher(cfg.TopicID) — NOT client.Topic()
    5. Set publisher.EnableMessageOrdering = true — BEFORE any Publish call
    6. Return &PubSubSinkConsumer{...}

    Deliver:
    1. Resolve topic: if topicT != nil, execute template against entry.Event; else use cfg.TopicID.
       (Note: Pub/Sub publisher is created for a fixed topicID at construction. TopicTemplate in Pub/Sub
       refers to the topic ID to use — unlike Kafka where the topic string is passed per-record.
       For simplicity in v2.6.0: if TopicTemplate is non-empty, the consumer must be created per-topic.
       Practical approach: ignore TopicTemplate in Deliver for now — publish always to the configured TopicID.
       TopicTemplate field is preserved in config for future multi-topic support. Document this limitation
       in a code comment.)
    2. orderingKey := string(entry.Event.Key)
    3. json.Marshal(entry.Event) → data
    4. start := time.Now()
    5. result := c.publisher.Publish(ctx, &pubsub.Message{Data: data, OrderingKey: orderingKey, Attributes: map[string]string{"Kaptanto-Idempotency-Key": entry.Event.IdempotencyKey}})
    6. _, publishErr := result.Get(ctx) — blocks until server ack (CHK-01)
    7. Observe latency: c.m.QueuePublishLatency.WithLabelValues("pubsub").Observe(time.Since(start).Seconds())
    8. If publishErr != nil:
       - var paused pubsub.ErrPublishingPaused; if errors.As(publishErr, &paused) { c.publisher.ResumePublish(paused.OrderingKey) }
       - c.m.QueuePublishErrors.WithLabelValues("pubsub").Inc()
       - return fmt.Errorf("pubsub sink: publish for key %q: %w", orderingKey, publishErr)
    9. c.m.QueuePublishTotal.WithLabelValues("pubsub").Inc(); return nil
    (Guard all metrics calls with `if c.m != nil`)

    Ping:
    ```go
    ctx, cancel := context.WithTimeout(context.Background(), 5*time.Second)
    defer cancel()
    _, err := c.client.TopicAdminClient.GetTopic(ctx, &pubsubpb.GetTopicRequest{
        Topic: fmt.Sprintf("projects/%s/topics/%s", c.projectID, c.topicID),
    })
    ```

    Close:
    ```go
    c.publisher.Stop()  // drain buffered messages first
    c.client.Close()    // then close gRPC connection pool
    ```

    ID: return c.id

    SetMetrics: c.m = m
